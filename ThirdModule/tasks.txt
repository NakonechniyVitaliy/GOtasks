
    Работа с файлами / Работа с JSON

✅ 1. Записать строку в файл
Создай файл output.txt и запиши в него текст "Hello, File!".

✅ 2. Прочитать содержимое текстового файла
Прочитай файл output.txt, выведи содержимое в консоль.

✅ 3. Записать структуру в файл как JSON
Создай структуру Person{Name string, Age int}, запиши слайс []Person в файл people.json в виде JSON.

✅ 4. Прочитать JSON из файла в структуру
Прочитай people.json из предыдущей задачи, распарсь его обратно в []Person, выведи содержимое.

✅ 5. Прочитать JSON из файла, изменить и сохранить обратно
Прочитай people.json, увеличь возраст каждому на 1, перезапиши файл.

✅ 6. Обработка файла, которого не существует
Попробуй открыть несуществующий файл nofile.txt, обработай ошибку и выведи “Файл не найден” вместо паники.

✅ 7. Добавить строку в конец файла
Добавь "New line" в конец output.txt, не перезаписывая старое содержимое.

✅ 8. JSON с вложенными структурами
Используй Person{Name, Age, Address}, где Address{City, Street}. Сохрани []Person в файл и снова прочитай.

✅ 9. JSON → map[string]interface{}
Прочитай JSON-файл без структур — в map[string]interface{}. Выведи ключи и значения.

✅ 10. Разделить на функции: saveToFile() / loadFromFile()
Напиши две универсальные функции:

func saveToFile(filename string, data any)
func loadFromFile(filename string, target any)
Они должны сохранять и читать JSON в любую структуру.
